dailyhoopstats uploader

A private desktop tool that publishes original NBA analysis Shorts to the Daily Hoop Stats channel on YouTube.

What it is

dailyhoopstats uploader is a command-line script used exclusively by the developer (who is also the channel owner) to upload finished, original short-form NBA analysis videos to their own YouTube channel and set each video's title, description and scheduled publish time.

What it is not

  • It is not distributed, sold or offered to anyone else — there are no end users.
  • It does not read, collect, store or share data about any YouTube user.
  • It does not comment, like, subscribe, message or interact with any other channel.
This application uses YouTube API Services (YouTube Data API v3, videos.insert) to upload videos. By using it, the user agrees to the YouTube Terms of Service and the Google Privacy Policy.
